# RecorderX

A modern, high-performance screen recording application for Android with a focus on privacy and high-quality output.

## Features

- **High-Resolution Recording:** Supports Native, 1080p, 720p, and 480p.
- **High FPS:** Up to 120 FPS recording (device dependent).
- **Internal Audio:** Capture system sound directly (Android 10+).
- **Modern Codecs:** Support for H.264 (AVC) and H.265 (HEVC).
- **Privacy Focused:** Works entirely offline. No internet permissions, no tracking.
- **Material 3 Design:** Clean, modern UI with glassmorphic elements.
- **Quick Settings Integration:** Start/Stop recording directly from your notification shade.

## F-Droid Checklist

- [x] **Fully Open Source:** Licensed under Apache 2.0.
- [x] **Privacy:** No internet permission, no tracking, no non-free dependencies.
- [x] **Standard Build System:** Uses standard Gradle build.

## Build Instructions

1. Clone the repository.
2. Open in Android Studio.
3. Run `./gradlew assembleDebug` to build the debug APK.
4. Run `./gradlew assembleRelease` to build the production APK.
